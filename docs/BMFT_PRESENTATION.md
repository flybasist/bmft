## 💡 Что это?

**BMFT (Bot Moderator For Telegram)** — ваш автоматический помощник для модерации и автоматизации Telegram-чатов.

### Что умеет бот:

- 🛡️ **Защищает** от флуда и спама автоматическими лимитами
- 📊 **Показывает** кто и как активен в чате (статистика за день/неделю)
- 🎭 **Развлекает** автоматическими реакциями на ключевые слова
- ⏰ **Напоминает** о важных событиях по расписанию (cron)
- 🚫 **Фильтрует** маты и нежелательный контент
- 🎯 **Работает с топиками** — разные настройки для каждой темы

**Главное:** Настраивается простыми командами прямо в чате — без программирования и серверов.

## ⚡ Главные возможности

### 📊 Статистика активности

Узнайте кто и насколько активен в чате:

```
/myweek     — ваша активность за последние 7 дней
/mystats    — ваши счётчики за сегодня
/topchat    — топ-10 самых активных (только админы)
/chatstats  — общая статистика чата (только админы)
```

**Что показывает:** сообщения, фото, видео, стикеры, войсы — всё раздельно по дням недели.

**Пример использования:**
- Найти самых активных участников для выдачи VIP
- Увидеть в какие дни чат наиболее живой
- Понять кто реально помогает сообществу

---

### 🛡️ Умные лимиты без жёстких банов

Установите ограничения на контент без потери участников:

```
/setlimit text 50       — максимум 50 текстовых сообщений в день
/setlimit sticker 10    — максимум 10 стикеров в день
/setlimit photo 20      — максимум 20 фото в день
/getlimit               — посмотреть текущие лимиты
```

**VIP-статус:** Ответьте на сообщение доверенного пользователя и отправьте `/setvip` — он получит иммунитет от лимитов.

**Как работает:**
- Превышен лимит → бот удаляет сообщение и мягко предупреждает
- Близко к лимиту → бот заранее предупреждает (порог настраивается)
- VIP-пользователи → пишут без ограничений
- Каждый видит свои счётчики через `/mystats`

**Реальный кейс:**
*В чате начался флуд стикерами. Вместо бана установили лимит 20 стикеров/день. Флуд прекратился, но участники остались.*

### 🎭 Автоматические реакции

Оживите чат пасхалками и полезными автоответами:

```
/addreaction привет [стикер]    — бот ответит приветственным стикером
/addreaction котики [фото]      — отправит фото котика
/addreaction "как дела" "Всё отлично! А у тебя?"
```

**Возможности:**
- Реакции на любые слова или фразы
- Любой тип ответа: текст, стикер, фото, GIF, видео
- Cooldown чтобы бот не спамил
- Персональные реакции на конкретного пользователя
- Работает и по тексту, и по подписям к фото/видео

**Реальный кейс:**
*Добавили реакцию на слово "документация" с ссылкой на wiki. Участники стали быстрее находить ответы, нагрузка на модераторов снизилась.*

### 🚫 Фильтр нежелательного контента

Автоматическая чистка без вашего участия:

**Встроенный фильтр матов:**
```
/setprofanity delete        — включить фильтр с удалением
/setprofanity delete_warn   — удалять и предупреждать
/profanitystatus            — проверить статус
```

**Кастомный фильтр слов:**
```
/addban спам delete             — запретить слово "спам"
/addban реклама delete_warn     — удалять рекламу с предупреждением
/addban http delete             — запретить любые ссылки
/addban спам|реклама|продам delete — несколько слов одной командой
```

**Автобан за мат:** При превышении лимита нарушений бот сначала предупреждает, затем банит. Порог настраивается.

**Реальный кейс:**
*Включили фильтр матов + забанили слово "реклама". Токсичность в чате упала на 80%, модераторы перестали жаловаться на перегруз.*

### ⏰ Задачи по расписанию

Автоматические напоминания и регулярные сообщения:

```

/addtask "0 18 * * 1-5" "Рабочий день окончен 🎉"   # Пн-Пт в 18:00
/addtask "0 12 * * 6,0" "С выходными! 🎈"           # Выходные в 12:00
```

Поддерживаются стикеры и фото — ответьте на стикер и добавьте задачу.

**Реальный кейс:**

## 🏗️ Под капотом

Для тех, кому интересна техническая сторона:

| Компонент | Технология |
|-----------|-----------|
| Язык | Go 1.25.5 |
| База данных | PostgreSQL 16 с партиционированием |
| Telegram API | telebot.v3 (Long Polling) |
| Инфраструктура | Docker, multi-stage build |
| Логирование | zap + lumberjack (ротация) |
| Планировщик | robfig/cron/v3 |

**Архитектура:** 5 модулей, pipeline из 3 middleware: `statistics → limiter → reactions`. Scheduler и Maintenance работают в фоне.

📖 **Подробная техническая документация:** [ARCHITECTURE.md](ARCHITECTURE.md) · [MODULES.md](modules/MODULES.md) · [DATABASE.md](architecture/DATABASE.md)

---

### Вариант 2: Self-hosted (полный контроль)

**Для тех, кто хочет:**
- Полный контроль над данными
- Модификацию кода под свои нужды
- Независимость от внешних сервисов

**Требования:**
- VPS с Docker (минимум 1GB RAM, 1 CPU)
- Базовые знания Linux и Docker
- 15 минут на установку

**Установка:**
```bash
git clone https://github.com/flybasist/bmft.git
cd bmft
cp .env.example .env
# Настроить TELEGRAM_BOT_TOKEN в .env
docker-compose -f docker-compose.env.yaml up -d
docker-compose -f docker-compose.bot.yaml up -d
```

📖 **Полная инструкция:** [QUICKSTART.md](QUICKSTART.md)
